============================
Simple Blogging Application
============================

**How to install code:**

1.git clone https://github.com/shitalp/blog
  
2.cd blog    
  
3.sudo python3 setup.py install

**How to run code:**
  
1.blog

**How to run test cases**

1.cd tests

2.wget https://raw.githubusercontent.com/lehmannro/assert.sh/master/assert.sh

3.bash tests.sh
